# Senior Project Fall 2025

This repo contains Crouton, an iOS mobile application which logs network data.

## Features

Socket layer app to network (and network to app) detection
Efficient background processes for the sniffer
Organize by most recent request or group by app
Delete all logs.
Tap on any log to gain more details about it
